Projeto 1: Página de Tributo - Charlie Brown Jr

Este projeto teve como objetivo criar uma página de tributo à banda brasileira de rock Charlie Brown Jr, destacando sua história, músicas de sucesso e impacto cultural.

Aspectos Relevantes do Projeto 1
Aspecto Descrição
Tecnologias Utilizadas HTML5 e CSS3 para estruturação e estilização da página.
Objetivo do Projeto Criar uma experiência imersiva para fãs e novos ouvintes da banda, explorando sua história e legado.
Desafios Enfrentados Organizar um grande volume de informações de maneira coesa e acessível, mantendo o design responsivo e atrativo.
Resultados Alcançados Uma página rica em conteúdo, com design responsivo que se adapta a diferentes dispositivos, melhorando a acessibilidade e a experiência do usuário.
Lições Aprendidas Importância da pesquisa e planejamento na organização do conteúdo; técnicas avançadas de CSS para design responsivo.

Projeto 2: Sistema de E-commerce com Spring Boot e JPA

Desenvolvimento de um sistema de e-commerce completo, abrangendo funcionalidades de gerenciamento de usuários, pedidos, produtos e categorias, utilizando Spring Boot e JPA para a construção de uma aplicação back-end robusta e eficiente.

Aspectos Relevantes do Projeto 2
Aspecto Descrição
Configuração de Ambiente de Desenvolvimento Configuração do perfil de teste no Spring Boot, facilitando o desenvolvimento e testes locais.
Modelagem de Entidades e Relacionamentos Modelagem de dados com entidades complexas e relacionamentos, utilizando JPA para ORM.
Gerenciamento de Dependências Uso do Spring Data JPA para gerenciamento de operações de banco de dados de forma declarativa.
Relações Complexas entre Entidades Configurações avançadas de relações entre entidades, incluindo chaves compostas.
Injeção de Dependências Aplicação do princípio de Injeção de Dependência para modularidade e testabilidade.
Implementação de Funcionalidades de Negócio Desenvolvimento de lógicas de negócio, como adição de itens a pedidos e cálculo do total.
Manipulação de Datas Uso de tipos de dados temporais para operações e transações.
Tratamento de Exceções e Validação de Dados Gerenciamento centralizado de erros e exceções para melhorar a experiência do usuário.
